## 📝 Customer Enquiry Form

The enquiry form captures:

- Customer Name
- Company Name
- Email Address
- Phone Number
- Location
- Product Interested In
- Service Required
- Message / Requirement

After submission:

- Customer details are securely stored.
- Confirmation message: **"Your enquiry has been submitted successfully."**
- Sales team receives the enquiry instantly.
